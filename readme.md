# BitTorrent协议中文翻译（p2p协议,p2p网络，对等网络）  
BEP 代表 BitTorrent 增强提案。BEP 是向 BitTorrent 社区提供信息或描述 BitTorrent 协议新功能的设计文档。BEP 应提供功能的简明技术规范和功能的基本原理。
